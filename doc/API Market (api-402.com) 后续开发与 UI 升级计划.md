# API Market (api-402.com) 后续开发与 UI 升级计划

## 1. 竞品拆解分析 (`claw402.ai`)

经过深入分析 `claw402.ai`，我们总结出其成功的几个核心要素，这也是后续 `api-402.com` 的发力点：

### 1.1 视觉与 UI/UX (纯正 Web3 未来感)
*   **极致的暗黑主题**：全站底色趋近于纯黑 (`#050505`)，而非沉闷的深灰。背景配有非常微弱的蓝色/灰色网格线 (Grid Lines)，营造出数字赛博空间的氛围。
*   **克制的高亮色彩**：文字主体为白色和浅灰，但在关键数据（如 USDC 价格）上使用了醒目的**霓虹青绿 (`#00FFAA`)**，在品牌词（如 USDC）和主打按钮上使用了**电蓝色 (`#0066FF`)**。
*   **终端风 (Terminal) 演示**：大量使用了模拟 macOS 终端风格的代码框，并且给代码框底部添加了极具视觉冲击力的**弥散发光阴影 (Glow Effect)**。这种 Show, Don't Tell 的方式极受开发者欢迎。
*   **卡片化布局**：API 列表使用了暗色背景、微边框的卡片，悬浮时有细腻的边框高亮或轻微的光晕变化。

### 1.2 核心业务逻辑
*   **完全去中心化身份**：抛弃传统账户体系 (No signup, No API key)，一切基于 EVM 钱包地址。
*   **EIP-3009 USDC 授权支付**：依靠 Base 链的低手续费，实现无感、高频的小额微支付 (Micro-payments)。
*   **定位清晰**：主打为 AI Agents 和开发者提供自动化的 API 购买渠道。

## 2. 后续开发路线图 (Roadmap)

为了让 `api-402.com` 达到甚至超越竞品水平，我们制定以下三个阶段的开发计划：

### 阶段一：前端视觉重铸 (立即执行)
*   **目标**：完全替换现有的渐变紫灰风格，转向极致的 Web3 暗黑黑客帝国/赛博朋克风。
*   **任务**：
    1.  引入背景网格动画和暗黑底色。
    2.  重写 Terminal 组件，增加代码打字机动画和**弥散光束背景**。
    3.  将 API 卡片的价格颜色统一改为极客风的荧光绿。
    4.  优化按钮交互，增加霓虹发光 Hover 效果。

### 阶段二：网关后端增强 (x402 协议深度集成)
*   **目标**：完善 Serverless 层的支付验证逻辑。
*   **任务**：
    1.  **链上验证接入**：在 Worker 中集成 Ethers.js 或 Viem 的精简版，真实验证前端传来的 EIP-3009 签名或交易 Hash，而不再是目前的 Demo 模式。
    2.  **API 请求代理与聚合**：网关不再仅仅返回 mock 数据，而是真实代理请求到下游 API（如 Binance, DeepSeek等），并在 Worker 侧处理鉴权和计费。
    3.  **并发与限流**：针对不同钱包地址设置频率限制，防 DDOS。

### 阶段三：生态工具链
*   **目标**：降低使用门槛。
*   **任务**：
    1.  提供前端 SDK (`@api-402/client`)，自动拦截 402 状态码调起钱包签名。
    2.  提供 Python 版本的 Agent SDK，方便 AI 自动充值并调用。
